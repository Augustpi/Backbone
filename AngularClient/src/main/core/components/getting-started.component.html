Getting Started

1. Create a new project with 'Todo App' template

2. Copy the following HTML block:

index.html
            
            {{ exampleCode }}

3. Navigate to Codepen and paste the copied block into HTML section

4. Voila! You created your first application on Backbone.
When you 'Complete' your items, your changes will be saved in Backbone.
If you wish, you can continue with updating your Todo App by adding or removing items
